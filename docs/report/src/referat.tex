Расчетно-пояснительная записка содержит 61 страницу, 57 рисунков, 1 таблицу, 22 источника, 2 приложения.

Ключевые слова: аэрофотоснимки, сверхразрешение, интерполяция, нейронные сети, коэффициент масштабирования, RCAN, SSIM, PSNR. 

Цель выпускной квалификационной работы: разработать метод сведения аэрофотоснимков с разными разрешениями к одному разрешению с использованием нейронной сети.

Для достижения поставленной цели требуется выполнить следующие задачи:

\begin{itemize}[label=---]
    \item провести анализ предметной области;
    \item рассмотреть существующие методы сведения аэрофотоснимков с разными разрешениями и сравнить их;
    \item разработать метод сведения аэрофотоснимков с разными разрешениями к одному разрешению с использованием нейронной сети;
    \item разработать программное обеспечение, реализующее описанный метод;
    \item исследовать зависимость качества изображения от коэффициента масштабирования.
\end{itemize}
