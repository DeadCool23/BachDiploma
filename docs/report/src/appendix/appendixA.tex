\chapter*{ПРИЛОЖЕНИЕ А}
\addcontentsline{toc}{chapter}{ПРИЛОЖЕНИЕ А}

Диск с рабочими материалами со следующей структурой папок:
\begin{verbatim}
  docs - каталог с РПЗ ВКР
  src/ - каталог с исходными кодами
    api - каталог с исходным кодом веб-сервера
    frontend/ - каталог с исходными кодами пользовательского
    интерфейса
      desktop - каталог с исходным кодом настольного приложения
    trained_models  - каталог с обученными нейронными сетями
    .env - конфигурационный файл
    Makefile - файл автоматизации сборки и запуска
\end{verbatim}
