\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

\textbf{Цель выпускной квалификационной работы достигнута:} разработан метод сведения аэрофотоснимков с разными разрешениями к одному разрешению с использованием нейронной сети.

В ходе выполнения данной выпускной квалификационной работы были решены следующие задачи:

\begin{itemize}[label=---]
    \item проведен анализ предметной области;
    \item рассмотрены существующие методы сведения аэрофотоснимков с разными разрешениями и сравнены;
    \item разработан метод сведения аэрофотоснимков с разными разрешениями к одному разрешению с использованием нейронной сети;
    \item разработано программное обеспечение, реализующее описанный метод;
    \item исследована зависимость качества изображения от коэффициента масштабирования.
\end{itemize}
