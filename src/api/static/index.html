ImgScaler API

Основные эндпоинты:

  • GET /api/v1/images/scale - Изменение разрешения изображения по заданному коэффициенту масштабирования
  • GET /api/v1/images/reduce - Сведение изображений с разными разрешениями к наивысшему разрешению

Технические эндпоинты:

  • /docs - интерактивная документация Swagger
  • /redoc - альтернативная документация ReDoc
